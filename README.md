# Python Pokedex App
Μια απλή εφαρμογή Pokedex που αναζητά και εμφανίζει πληροφορίες απο Pokemon από ένα αρχείο JSON.

## Χαρακτηριστικά
### Ολες οι πληροφορίες αντλήθηκαν απο το PokeAPI (https://pokeapi.co/)
Αναζήτηση Pokemon με βάση το όνομα και το ID. 
Εμφάνιση στατιστικών (HP , Attack , κλπ).
Διαχείριση δεδομένων μέσω αρχείου 'pokemons.json'

## Τεχνολογίες 
* Γλώσσα: Python 3.x
* Data: Json
  
## Εγκατάσταση και χρήση 
1. Κατεβάστε το repository:
   'git clone https://github.com/greybea/myPokedex'
2. Τρέξτε την εφαρμογή:
    'python pokedex.py'
